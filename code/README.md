# 実行の仕方

## 環境
Linux

## 準備
```sh
mkdir client/bin                                                                                                                          
mkdir server/bin                                                                                                                          
```

## ビルド
```sh
cd client && make                                                                                                                         
cd ../server && make
```

## 実行
`client/bin/`
`server/bin/`

にそれぞれ実行ファイルができていると思います。
